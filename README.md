# ScriptForge

**ScriptForge** es una biblioteca portátil de scripts que te permite guardar, documentar y recuperar tus scripts en segundos desde cualquier lugar, directamente en el navegador, sin servidores ni dependencias externas.

---

## Características

- **Agregar scripts** con nombre, lenguaje, descripción y código fuente
- **Pasos e indicaciones** opcionales por script (hasta 20 pasos configurables)
- **Biblioteca filtrable** con búsqueda en tiempo real por nombre, lenguaje o descripción
- **Probador integrado** para ejecutar scripts JavaScript directamente en el navegador
- **Respaldo y restauración** del historial de scripts en formato JSON
- **Importar / Exportar** la biblioteca completa
- **Resaltado de sintaxis** con Prism.js para múltiples lenguajes
- **Sin backend** — todo se almacena en `localStorage`

## Uso

1. Abre `index.html` en cualquier navegador moderno (no requiere servidor).
2. Ve a **Agregar script** y completa el formulario.
3. Consulta tu biblioteca en la sección **Biblioteca**.
4. Prueba scripts JavaScript desde el **Probador**.
5. Genera respaldos en la sección **Respaldo** para no perder tu colección.
